# 📖 Como Usar

Guia passo a passo para aproveitar ao máximo as personas.

## Passo 1: Escolha
Selecione a persona que melhor se adequa à sua necessidade.

## Passo 2: Cole
Copie o prompt completo e cole no ChatGPT.

## Passo 3: Customize
Adapte para seu contexto específico:
- Substitua `[variável]` pelos seus dados
- Ajuste o tom conforme necessário
- Refine com feedback

## Passo 4: Aproveite
Use repetidamente. Quanto mais refinar, melhor!

## 💡 Dicas Profissionais

1. **Use em novo chat** para melhor performance
2. **Combine personas** para resultados mais ricos
3. **Refine com feedback** do ChatGPT
4. **Reutilize** suas variantes favoritas
