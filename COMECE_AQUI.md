---

## Passo 2: Copie o Prompt

1. Abra a pasta correspondente (ex: `01-personas/`)
2. Clique no arquivo do especialista (ex: `dr-data-pro.txt`)
3. Clique em "Copy raw contents" ou selecione tudo (Ctrl+A) e copie
4. **Copie TODO o conteúdo** - inclui instruções importantes!

---

## Passo 3: Abra ChatGPT

1. Vá para https://chat.openai.com
2. Clique em "New chat"
3. **Cole o prompt completo**
4. Pressione Enter

---

## Passo 4: Customize (Opcional)

Os prompts têm placeholders para você customizar:

```
[seu_nome] → Seu nome real
[contexto] → Seu contexto específico
[objetivo] → Seu objetivo
[dados] → Seus dados/informações
```

**Exemplo:**
```
Antes: "Você é um especialista em [área]"
Depois: "Você é um especialista em Data Science para Startups"
```

---

## Passo 5: Comece a Conversa!

Depois que o ChatGPT recebe o prompt, ele ativa o modo especialista.

Agora você pode:
- ✅ Fazer perguntas
- ✅ Pedir análises
- ✅ Solicitar exemplos
- ✅ Explorar tópicos profundos

---

## 💡 Dicas Valiosas

### 1️⃣ Use em Chat Novo
Cada persona funciona melhor isolada em um chat dedicado.

### 2️⃣ Não Edite Demais
Os prompts foram testados. Customize apenas o mínimo necessário.

### 3️⃣ Leia as Instruções
Cada prompt tem seção "Como usar" - leia!

### 4️⃣ Refine com Feedback
Se não gostou? Peça "Responda de forma [mais técnica/simples/criativa]"

### 5️⃣ Combine Prompts
Quer fusion de especialistas? Use output de um como input de outro!

## ❓ Dúvidas Frequentes

**P: Funciona com ChatGPT Free?**
A: Sim! Mas o Plus é mais rápido.

**P: Posso usar em outros modelos?**
A: Sim! Claude, Gemini, etc também funcionam.

**P: Preciso pagar para usar?**
A: Não! Só precisa de uma conta ChatGPT (gratuita).

**P: Como faço um prompt mais curto?**
A: Remova as seções "Exemplos" ou "Contexto Adicional".

**P: Funciona em português?**
A: Sim! Todos os prompts estão em PT-BR.
