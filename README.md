## 📖 Como Usar

### 1. Escolha uma Persona

```
Vá até 01-personas/ e escolha o especialista que precisa
Exemplo: Dr. Data Pro para Data Science
```

### 2. Copie o Prompt Completo

```
Abra o arquivo .txt
Copie TODO o conteúdo (incluindo instruções)
```

### 3. Cole no ChatGPT

```
1. Abra https://chat.openai.com
2. Novo chat
3. Cole o prompt completo
4. Customize conforme necessário
5. Comece a conversa
```

### 4. Customize

```
Substitua [seu_contexto] pelas suas informações
Ajuste tone, estilo e profundidade
Refine conforme feedback
```

## 💡 Dicas de Uso

### Para Máxima Eficiência

1. **Use em Chat Novo** - Cada persona funciona melhor isolada
2. **Leia Instruções Completas** - Há contexto importante nos prompts
3. **Customize Variáveis** - Substitua `[nome]`, `[contexto]`, etc.
4. **Teste Iterativamente** - Refine o prompt com feedback
5. **Combine com Referências** - Adicione seus documentos/dados

### Truques Avançados

- 🔄 **Chain Prompts** - Use output de um como input de outro
- 🎯 **Sistema de Roles** - Persona + Framework = superpoder
- 📊 **Análise Estruturada** - Combine com templates JSON
- 🔗 **Multi-Step Workflows** - Crie fluxos de trabalho completos
